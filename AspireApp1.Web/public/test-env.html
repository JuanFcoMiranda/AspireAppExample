🔍 Test de Variables de Entorno - OpenTelemetry/Aspire

⚠️ Instrucciones:

Este archivo sirve para verificar que las variables de entorno se están inyectando correctamente desde Aspire.

  1. Si ejecutas desde Aspire AppHost, las variables deben aparecer en window.*
  2. Si ejecutas con pnpm dev, las variables deben aparecer en import.meta.env.VITE_*

🌐 Variables en window (inyectadas por Aspire)

⚙️ Variables en import.meta.env (desde .env)

Nota: Este test está en HTML estático y no tiene acceso a import.meta.env.

Para verificar estas variables, abre la consola del navegador en tu aplicación Vue.js y ejecuta:

console.log(import.meta.env.VITE_OTEL_EXPORTER_OTLP_ENDPOINT);
console.log(import.meta.env.VITE_OTEL_SERVICE_NAME);
console.log(import.meta.env.VITE_OTEL_SERVICE_VERSION);

✅ Resultado esperado

OTEL_EXPORTER_OTLP_ENDPOINT:
http://localhost:4318/v1/traces
(o el puerto configurado en tu Aspire Dashboard)
OTEL_SERVICE_NAME:
aspireapp1-web
(o el nombre configurado en tu AppHost)
OTEL_SERVICE_VERSION:
0.1.0
(opcional)
